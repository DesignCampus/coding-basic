# DESIGN CAMPUS 教材

デザキャンの環境構築の教材です。  

## ファイル構成

```
├ [doc] : 講義についてのドキュメント
│   ├ [img]：ドキュメントで使用する画像を格納
│   ├ doc-Xw.md：各週の講義のスライド
│   └ my-theme.css：スライド用CSS
│
├ [example] : 講義で使ったサンプル
│ ├ assets：HTML以外の様々なファイルを格納
│ | ├ css：CSSを格納
│ | ├ images：画像ファイルを格納
│ | └ js：JavaScriptを格納
| |
| └ index.html
│
└ README.md : このファイル
```
